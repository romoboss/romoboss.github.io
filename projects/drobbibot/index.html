DrobbiBot

DrobbiBot is a Discord Bot based on fun focused commands, created using DSharpPlus and Visual Studio.

Commands

/jiggle
/joke
/combine
/cat
/dog
/coinflip
/randomnumber
/trivia
And more...

Hosting

The Bot is currently not being hosted publicly!
In order to use this bot you need to have a bot token. To get a token you neeed to create your own bot in the Discord Developer portal, that can be found here

Use Instructions

1. Download Visual Studio (or Visual Studio Code) if you haven't already
2. Download Program.cs and DrobbiBot.csproj
3. Put the files in a folder
4. Open the folder using Visual Studio
5. Run Program.cs
